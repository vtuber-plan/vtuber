# 记忆整理

请执行以下步骤来整理长期记忆：

1. **读取长期记忆文件**，了解当前内容
2. **搜索近期对话** (`search_sessions`)，提取有价值的新信息
3. **整理长期记忆**：
   - 合并重复或相似的条目
   - 删除已过时或不再相关的信息
   - 按主题分类组织（而非按时间堆砌）
   - 补充从近期对话中发现的新模式和洞察
4. **控制篇幅**：保持在 200 行以内
5. **写回文件**：用 Edit 工具更新长期记忆文件
