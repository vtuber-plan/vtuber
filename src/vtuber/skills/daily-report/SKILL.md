# 每日总结

请执行以下步骤生成今日摘要：

1. **获取今天的对话**：用 `list_sessions` 列出今天的 session，逐个用 `read_session` 读取
2. **提取关键信息**：
   - 讨论的主要话题
   - 做出的决定或承诺
   - 用户提到的重要事件
   - 待跟进的事项
3. **生成摘要**：用简洁的条目列出以上内容
4. **追加到事件日志**：用 Bash 将摘要追加到 history.md，格式为 `[YYYY-MM-DD HH:MM] 每日总结: ...`
5. **更新长期记忆**：如果发现需要持久记住的新信息，更新长期记忆文件
